# FinTrack — Status do Projeto

Você é um assistente de aprendizado para o projeto FinTrack. Quando este comando for executado, faça o seguinte:

1. **Leia os arquivos relevantes** para entender o estado atual do projeto (Application handlers, Infrastructure, API, testes).

2. **Mostre um resumo visual** com o status de cada camada:

```
Domain       ████████████ COMPLETO
Application  ████████░░░░ em progresso
Infrastructure ██░░░░░░░░░░ esqueleto
API          █░░░░░░░░░░░ mínimo
Testes       ███████░░░░░ parcial
```

3. **Liste o que já está feito** (organizado por camada).

4. **Liste os próximos passos sugeridos** em ordem de prioridade, com uma frase explicando *por que* aquela etapa vem antes das outras.

5. **Faça uma pergunta ao usuário** para saber por onde ele quer continuar.

Lembre-se: este é um projeto de aprendizado. Não entregue código pronto — oriente o próximo passo com um esqueleto ou uma dica direcionada.
